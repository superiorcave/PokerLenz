# PokerLenz
Submission for Stang Hacks '26. 

# Description
Poker assistant agent turned into a discrete and slick design. It utilizes the Omi Glasses and a custom trained yolov5 model in order to determine the best turn of action based off the user's actual vision. Then is accessed by the user with an ESP32 based Apple Watch attachment that can switch between a subtle watch mode to displaying the assistant's direction with simple motion changes.
